COMANDOS GIT

git status => USADO PRA SABER O QUE ESTÁ ACONTECENDO NO PROJETO

git init => CRIA UMA PASTA .GIT DENTRO DA PASTA DO PROJETO PARA O MONITORAMENTO DA PASTA. OBS: NÃO MEXER NA PASTA

git status => VERIFICA SE TEM ALGUM ARQUIVO PRA ADICIONAR DENTRO DO GIT

git add . => ADICIONA OS ARQUIVOS QUE AINDA NÃO FORAM ADICIONADOS DENTRO GIT

git commit -m => "DEIXE UMA MENSAGEM SOBRE O ARQUIVO"

git remote add origim => ENDEREÇO DA PASTA CRIADO NO GITHUB PARA ONDE VAI SEU REPOSITORIO (LIGANDO O REPOSITORIO LOCAL COM O REMOTO NO GITHUB)

git push -u origin main ou master => envia seu repositório local para o github
